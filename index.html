🔑 Il manque la clé Google Maps

L'app a besoin d'une clé API Google (gratuite dans votre usage) pour la carte, les photos et les temps de trajet en transports.

  1. Va sur console.cloud.google.com, crée un projet
  2. Active Maps JavaScript API, Places API et Directions API
  3. Crée une clé API (Identifiants → Créer → Clé API)
  4. Restreins-la à ton domaine GitHub Pages (Restrictions → Sites web)
  5. Colle-la dans index.html ligne ~où il y a GOOGLE_MAPS_API_KEY

Recharge la page ensuite — tout s'allume. ⚡
